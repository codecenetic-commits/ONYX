# Security Policy

## Supported Versions

Currently, the following versions of ONYX are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of ONYX seriously. If you believe you have found a security vulnerability in this project, please report it to us by emailing [support@onyx-watches.com].

Please do not report security vulnerabilities through public GitHub issues.

Once we receive your report, we will:

1.  Acknowledge receipt of your report within 48 hours.
2.  Provide a timeline for addressing the issue.
3.  Notify you once the vulnerability is fixed.
4.  Publicly disclose the vulnerability and credit you for the find, if desired.
