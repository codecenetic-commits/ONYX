## 🚀 How to Deploy

### Quick Start (Vercel - Recommended)
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy production
vercel --prod
```

### With Docker
```bash
# Build image
docker build -t onyx-watches .

# Run container
docker run -p 3000:3000 onyx-watches
```

### Local Testing
```bash
# Build
npm run build

# Test locally
npm start

# Visit http://localhost:3000
```
